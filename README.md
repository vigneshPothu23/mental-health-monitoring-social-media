\# Mental Health Monitoring on Social Media Using Emotion and Sarcasm Detection



\## Overview

This project detects emotions and sarcasm from social media text and

combines both predictions using a fusion inference layer.



\## Modules

\- Emotion Detection

\- Sarcasm Detection

\- Fusion Inference

\- Flask-based application



\## Note

Datasets, training splits, and trained models are intentionally excluded

from this repository.
